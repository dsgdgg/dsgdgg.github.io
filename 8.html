如何將網站放到 GitHub Pages

步驟 1：建立 GitHub Repository 步驟 2：上傳網站檔案 步驟 3：啟用 GitHub Pages

步驟 1：建立 GitHub Repository

  1. 登入你的 GitHub 帳戶。
  2. 點擊右上角的 + New repository 按鈕。
  3. 輸入 Repository 名稱（例如：my-website）。
  4. 選擇 Public（公開），然後點擊 Create repository。

步驟 2：上傳網站檔案

  1. 進入剛剛建立的 Repository。
  2. 點擊 Add file，然後選擇 Upload files。
  3. 將你的網站檔案（例如 index.html 和相關資源）拖曳到上傳區域。
  4. 填寫提交訊息，然後點擊 Commit changes。

步驟 3：啟用 GitHub Pages

  1. 進入 Repository，點擊 Settings。
  2. 滾動到 Pages 部分。
  3. 在 Source 下拉選單中選擇 Main（或你的主分支）。
  4. 點擊 Save。
  5. 稍等片刻，你的網站將可以在 https://[你的 GitHub 使用者名稱].github.io/[repository 名稱] 訪問。
